O meu projecto final de Síntese Sonoro para Media Digitais é a criação de uma pedaleira em Pure Data para uma guitarra.
A pedaleira pode seleccionar um ou vários efeitos. 
Os efeitos disponíveis são Reverb, Delay, Overdrive(distorção), o efeito wah, vibrato e tremolo. Alguns efeitos podem ter ser modificadas as características de acordo com a preferência ou procura do utilizador.
A configuração do pure data tem como input o canal da guitarra através de uma interface virtual ou de uma placa de som.
